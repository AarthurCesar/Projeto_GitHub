Comando git checkout

O comando git checkout é usado principalmente para trocar de branch ou restaurar arquivos em versões anteriores.

🌱 Criar e trocar para uma nova branch

Você pode criar e já entrar nela com o comando:

git checkout -b nova-branch

🔁 Trocar de branch

git checkout nome-da-branch

Você muda para a branch especificada. Exemplo:

git checkout nome_da_branch

⏪ Restaurar arquivo específico

Restaura um arquivo para a última versão confirmada (commit):

git checkout -- caminho/do/arquivo

⚠️Atenção: Isso apaga mudanças locais nesse arquivo!

📌 Alternativa moderna

Hoje, o Git recomenda o uso de:
  • git switch nome-da-branch → para trocar de branch
  • git restore arquivo → para restaurar arquivo

⚠️ Dicas importantes

  • Salve seu trabalho antes de trocar de branch com git add e git commit.
  • Evite trocar de branch com arquivos modificados não salvos.
