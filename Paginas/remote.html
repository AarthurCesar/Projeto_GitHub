Comando git remote

O comando git remote é usado para conectar o repositório local a um repositório remoto, como o GitHub. Ele também pode ser usado para listar, renomear ou remover conexões remotas.

🔗 Adicionar um repositório remoto

Para conectar seu projeto local ao GitHub:

git remote add origin https://github.com/usuario/nome-do-repositorio.git

  • origin → nome padrão do repositório remoto
  • URL → link HTTPS ou SSH do seu repositório no GitHub

📋 Verificar repositórios remotos conectados

git remote -v

Mostra os links configurados para fetch (buscar) e push (enviar).

✏️ Renomear um repositório remoto

git remote rename origin upstream

Renomeia a conexão remota de origin para upstream.

🗑 Remover um repositório remoto

git remote remove origin

Remove a conexão remota chamada origin.
