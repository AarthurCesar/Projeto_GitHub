Comando git branch

O comando git branch serve para gerenciar diferentes ramos (branches) dentro de um repositório Git. Ele permite criar, listar e excluir branches.

📄 Ver todas as branches

Mostra uma lista das branches locais. A branch atual estará com um asterisco *.

git branch

🌱 Criar uma nova branch

Cria uma nova branch, mas você ainda continuará na branch atual.

git branch nome-da-branch

🔄 Trocar de branch

Use git checkout ou o atalho moderno:

git switch nome-da-branch

🔥 Excluir uma branch

git branch -d nome-da-branch

Exclui uma branch que já foi mesclada. Se não foi, e quiser forçar:

git branch -D nome-da-branch

🌍 Branch remota

Para ver branches remotas:

git branch -r

🧠 Dica: Branch vs Checkout

  • git branch → cria ou lista branches
  • git checkout ou git switch → muda para outra branch

⚠️ Boas práticas

  • Dê nomes descritivos para as branches, como correcao-login ou nova-feature.
  • Antes de criar uma branch, atualize sua branch atual com git pull.
