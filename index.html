Comandos Básicos para GitHub

1 - git config

Define seu nome e e-mail como autor dos commits.

git config --global user.name "Seu Nome"

git config --global user.email "seu@email.com"

2 - git init

Cria um novo repositório Git na pasta atual.

git init

3 - git status

Mostra quais arquivos foram modificados ou adicionados desde o último commit.

git status

4 - git add

Adicionar um arquivo específico

Prepara o arquivo para ser incluído no próximo commit.

git add nome_do_arquivo

Adicionar todos os arquivos

Adiciona todos os arquivos modificados para o commit.

git add .

5 - git commit

Salva uma versão do projeto com uma mensagem explicando as mudanças.

git commit -m "mensagem do commit"

6 - git remote

Conecta seu projeto local com o repositório do GitHub.

git remote add origin https://github.com/usuario/repositorio.git

7 - git push

Envia seus commits para o repositório remoto.

git push -u origin main

8 - git pull

Baixa e mescla as alterações do repositório remoto com o seu local.

git pull origin main

9 - git clone

Clona um repositório remoto para o seu computador.

git clone https://github.com/usuario/repositorio.git

10 - git branch

Lista os branches ou cria um novo branch.

git branch

git branch nome-do-branch

11 - git checkout

Troca de branch ou restaura arquivos.

git checkout nome-do-branch

12 - git merge

Combina as alterações de um branch com outro.

git merge nome-do-branch
